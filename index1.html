HTML Terms and Definitions

Here are terms and definitions:

DOCTYPE Statement
It is information that helps the browser understand what type of document is expected.
How to use "<ol>";"<ul>";"<dl>"
To use the ordered list element , you must type the ordered list open tag then add in the open list tag to proceed. Once done, then you add the closed list tag for each list item. Finally then you add the closed ordered list tag. The same instructions follow for an unordered list element. To use the definition list element, you must first add in the open definition list tag. Then you add in your terms by using the define terms tag ("<dt>"). To define the term you add the describe term open tag ("<dd>"). Then add closed tags for each open tag section.
Difference between a text editor and word processor
A text editor has plain text where you can add, remove, or change any text/code/characters within a file. You cannot edit or format text, only add,remove, or change text. A word processor uses options and commands to format and style text within a document.
Client-Server Model
A client-server model is an application that will work between the providers of a service, which is a server, and the service requester, which is a client.
FTP
FTP stands for File Transfer Protocol, and it has rules to tell the computer how to transfer files from a system to another through the internet.
HTTP
HTTP stands for Hypertext Transfer Protocol. It is a protocol that fetches resources and documents.
TCP/IP
TCP/IP stands for Transmission Control Protocol/Internet Protocol. It sets rules for the computers to communicate with the internet/network.
